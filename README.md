# Real-Time Chat App

A simple real-time chat application using Next.js, .Net Core and websockets for real-time updates.

# How it works

1. Create an account.
2. Open your contacts.
3. Write and receive messages.
4. Add an email to your account for email notifications.

# Features

- .Net Core 9
- Entity Framework
- Identity
- SignalR
- Next.js
- Postfix
- Dovecot
- TLS
